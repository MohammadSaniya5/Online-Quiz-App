#  Online Quiz Application 

A responsive and interactive Online Quiz Platform developed during my IBM SkillsBuild Front-End Web Development Internship.  
This project focuses on clean UI design, smooth navigation, and real-time quiz evaluation using core web technologies.

---

##  Tech Stack

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- GitHub Pages (Deployment)

---

##  Key Features

-  Multiple subject quizzes  
-  Instant result calculation after submission  
-  Fully responsive design (Mobile & Desktop friendly)  
-  Clean navigation and user-friendly interface  
-  Deployed using GitHub Pages
